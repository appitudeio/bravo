## Built-in loading states

The main enhancement is automatic loading state management — no more manual spinner handling:

<div class="bd-example mb-3">
  <button type="button" class="btn btn-primary me-2" onclick="this.showLoader(); setTimeout(() => this.hideLoader(), 2000)" data-bs-loader="Saving…">
    Click to test loading
  </button>
  <button type="button" class="btn btn-success" onclick="this.showLoader(); setTimeout(() => this.hideLoader(), 2000)" data-bs-loader>
    Auto-reset (2s)
  </button>
</div>

```html
<!-- Shows spinner + "Saving…" text when triggered -->
<button type="submit" class="btn btn-primary" data-bs-loader="Saving…">
  Save
</button>

<!-- Shows spinner only (with SR-friendly "Loading…") -->
<button type="submit" class="btn btn-primary" data-bs-loader>
  Submit
</button>
```

**JavaScript API:**
```js
const btn = document.querySelector('#save');
btn.showLoader();   // Disables button, shows spinner
// ... async work ...
btn.hideLoader();   // Restores original state
```

## New utilities

**`.btn-default`** — A neutral button that automatically adapts to light/dark themes (no more manually switching between btn-light and btn-dark).

**`.btn-xs`** — Extra small size for dense UIs like data tables or toolbars.

**`.btn-icon`** — Square aspect-ratio button that automatically matches the height of regular buttons (no manual width/height CSS needed).

**`.btn-rounded`** — Pill-shaped buttons. Combine with `.btn-icon` for perfect circles.

## Details

* **Loading states**: Automatically disable button, show spinner, handle ARIA attributes
* **Dark mode aware**: Hover/active states adjust for visibility
* **Focus rings**: Clean outline styling replaces Bootstrap's box-shadow

<div class="alert alert-info mt-4">
  <strong>Auto-initialization:</strong> Buttons with <code>data-bs-loader</code> are automatically enhanced when added to the DOM, even dynamically. This is powered by Bravo's <a href="/docs/0.0/javascript/dynamic-observer/">DynamicObserver</a>.
